- WTForms

- Bootstrap

## Running

Setup virtual environment:

```
$ virtualenv -p /usr/bin/python3 venv
```

Install pip requirements

```
$ pip install -r requirements.txt
```

Update `config.py` with your database settings and then:

```
$ ./run.py
```
